Το πρόγραμμα περιλαμβάνει τα αρχεία:

lsh.c			Η main συνάρτηση του προγράμματος που καλεί την αντίστοιχη συνάρτηση ανάλογα με το αρχείο εισόδου 
cosine_eucl.c		Η υλοποίηση των απαράιτητων συναρτήσεων για τις περιπτώσεις euclidean και cosine
hash.c			Υλοποίηση συναρτήσεων για την δημιουργία και γέμισμα του hashtable
Cstruct.c		Υλοποιηση συναρτήσεων για τη δημιουργία των αλγορίθμων τυχαίων αριθμών για τις περιπτώσεις euclidean και cosine
metric.c		Η υλοποίηση των απαράιτητων συναρτήσεων για τη περίπτωση matrix distances
hamming.c		Η υλοποίηση των απαράιτητων συναρτήσεων για τη περίπτωση hamming
functions.c		Υλοποίηση βοηθητικών συναρτήσεων για τις hamming,metric και hash
hash.h			Δήλωση όλων των συναρτήσεων που χρησημοποιούνται στο πρόγραμμα
records.h		Δήλωση όλων των δομών τους προγράμματος
Makefile		


το πρόγραμμα μεταγλωτίζεται από το Makefile με ένα make ή με τη χρήση της εντολής 
gcc -o lsh lsh.c cosine_eucl.c hash.c Cstruct.c metric.c hamming.c functions.c -lm

και εκτελείται με την εντολή:
./lsh -k (k) -L (L) -d (datafile) -q (queryfile) - o (outputfile)
με τα k,L να είναι optional και τα files είτε να βρίσκονται στον ίδιο φάκελο με το εκτελέσιμο είτε να δίνεται το absolut path σε αυτά

στην περίπτωση cosine/euclidean ο χρήστης πρέπει να πληκτρολογει όταν του ζητηθεί στην γραμμή εντολών για ποια από τις δύο μεθόδους πρόκειται
επειδή τα Datafiles είναι τα ίδια και για τα δύο
